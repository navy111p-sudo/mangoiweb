👋 본문 영역 (Body Content)

이 영역은 사이드바가 열려 있어도 그대로 보이고 클릭/스크롤이 가능합니다.

🎨 디자인 설명

사이드바는 backdrop-filter: blur(18px) 로 흐릿한 반투명 효과.
뒤 콘텐츠 색이 살짝 비쳐 보입니다.

🔑 핵심 기술 — pointer-events: none

오버레이 div 에 pointer-events: none 를 적용하면 사이드바가 열려도 본문 클릭·스크롤이 그대로 작동합니다.

아직 클릭 안 함

📋 동작 방법

• 좌상단 파란 ☰ 탭 → 사이드바 슬라이드 인 (버튼이 🔴 ✕ 로 변경)
• 다시 ✕ 탭 → 사이드바 슬라이드 아웃
• 사이드바 열린 상태에서도 본문 영역 클릭 가능

📖 더 많은 콘텐츠

스크롤 가능 — 사이드바가 열려도 본문 스크롤이 작동합니다.
