# 경영 브리핑 카카오 알림톡 — 정식 발송 설정 안내

> 매일 오전 9시(KST) 학생수·매출·비용 브리핑을 카카오 알림톡으로 발송합니다.
> 코드·서버·대시보드는 **모두 준비 완료**. 아래 계정·검수 절차만 끝내고 시크릿을 넣으면 자동 발송됩니다.

## 지금 상태 (요약)
- ✅ 서버 자동발송 코드: Cloudflare Worker cron(매일 09:00 KST)이 `exec_recipients` 수신자에게 발송
- ✅ 수신자 등록: `010-8988-2224`(대표) — 등록 완료
- ✅ 대시보드: https://mango-i.com/admin/exec (대리점별 격리 적용)
- ⛔ **SOLAPI 미설정** → 아직 실제 발송 안 됨. 아래 3단계 필요.
- 그 전까지는 매일 9시 카카오톡 '나와의 채팅'으로 받는 백업이 계속 동작합니다.

---

## 1단계 — SOLAPI 계정 + 발신번호 (사장님 직접)
1. https://solapi.com 가입 + 본인인증
2. **발신번호 등록**(010-8988-2224 인증) — 문자 폴백/알림톡 발신에 필요
3. API 키 발급: 콘솔 → 개발/연동 → **API Key** 생성 → `API Key` + `API Secret` 복사

## 2단계 — 카카오 비즈채널 + 알림톡 템플릿 검수 (1~2일)
1. 카카오 비즈니스(business.kakao.com)에서 **망고아이 채널** 개설
2. SOLAPI에 카카오 채널 연동 → **발신프로필 ID(PFID)** 발급
3. 아래 템플릿을 **알림톡 템플릿**으로 등록·검수 신청:

**템플릿명:** 경영_일일브리핑
**템플릿 내용(그대로 복사):**
```
[망고아이 경영 브리핑] #{날짜}

▶ 재원생 #{재원생}명 (이번달 신규 +#{신규})
▶ 어제 매출 #{어제매출}
▶ 어제 비용 #{어제비용}
▶ 어제 순익 #{어제순익}
▶ 이번달 매출 #{월매출} / 비용 #{월비용} / 순익 #{월순익}

자세한 차트는 아래 버튼에서 확인하세요.
```
**버튼:** [웹링크] 버튼명 `대시보드 열기` → `https://mango-i.com/admin/exec`

> 검수 통과되면 **템플릿 코드**가 나옵니다(예: `EXEC_BRIEFING_01`).

## 3단계 — Worker 시크릿 설정 (제가 도와드릴 수 있음)
`cloudflare-deploy` 폴더에서 아래 명령으로 시크릿 등록(값 입력창이 뜸):
```
npx wrangler secret put SOLAPI_API_KEY
npx wrangler secret put SOLAPI_API_SECRET
npx wrangler secret put SOLAPI_PFID
npx wrangler secret put SOLAPI_FROM_PHONE          # 예: 01089862224
npx wrangler secret put SOLAPI_TEMPLATE_EXEC_BRIEFING   # 2단계의 템플릿 코드
```
등록 후 `deploy.bat` 1회 실행하면 다음날 09:00부터 자동 발송됩니다.

> 검수 전이라도 1~2단계(API키+발신번호)만 끝내면, 알림톡 실패 시 **문자(SMS)로 자동 폴백**됩니다.

## 테스트 (설정 후)
관리자 로그인 상태에서 아래를 호출하면 즉시 1회 발송 테스트:
```
POST https://mango-i.com/api/admin/exec/send-briefing
```
결과 JSON의 `mode`가 `live`(또는 `mock`)면 정상, `disabled`면 아직 키 미설정입니다.
