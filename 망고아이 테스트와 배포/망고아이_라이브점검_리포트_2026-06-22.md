## ⚠️ 발견된 이슈 (심각도 순)

### 1. [중] 우측 상단 "검은 빈 박스" (동영상 미로딩)
- 홈/메뉴 화면 우측 상단에 정체 모를 **검은 사각형(약 237×237)** 이 표시됨.
- 정체: `<video src="/video/langedu-female.mp4">` 요소. `readyState=0`(데이터 미로딩) 상태로 재생 시도 중이나 화면엔 빈 검정만 보임.
- 파일은 HTTP 200이지만 `content-length` 헤더가 비어 있어(다른 동영상은 정상 표기), 파일 손상/0바이트 또는 스트리밍 설정 문제 의심.
- **권장**: 해당 mp4 파일 정상 여부 확인 후 교체하거나, 미로딩 시 비디오 박스를 숨기는 폴백(poster 이미지 또는 display:none) 추가.

### 2. [하] 문구 오타 — "전체 매뉴"
- 전체메뉴 오버레이 헤더가 **"전체 매뉴"** 로 표기됨 → 올바른 표기는 **"전체 메뉴"**.
- **권장**: 텍스트 한 곳 수정.

### 3. [하] 존재하지 않는 경로가 404 대신 200(index.html) 반환
- 예: `/this-path-should-404-xyz`, 잘못된 `/api/*` 경로 호출 시 **HTTP 200 + index.html** 반환(SPA catch-all).
- SPA에선 흔한 동작이나, 잘못된 API 호출의 실패가 가려져 디버깅이 어려워질 수 있음.
- **권장**: 실제 존재하지 않는 `/api/*` 경로는 404(JSON)로 응답하도록 라우팅 분기.

### 4. [하·확인필요] 좌측 사이드 드로어 데스크톱 동작
- 좌측 가장자리 "열림/닫힘" 탭 클릭 시 버튼 라벨 상태는 바뀌나, 데스크톱(2560px)에서 드로어 패널(`.mg-drawer`)이 화면 밖(`translateX(-325px)`)에 그대로 남음.
- 본 앱은 **모바일·가로모드 우선 설계**라 드로어가 모바일 전용일 가능성이 높음 → **실제 모바일/APK 환경에서 추가 확인 필요**(데스크톱 한정 현상일 수 있어 단정 보류).
